
Canyon Tutors - Starter Website
--------------------------------
Open the folder in VS Code and launch index.html with Live Server to preview.
Files included:
- index.html, about.html, team.html, testimonials.html, act.html, resources.html, contact.html, faq.html
- style.css, logo.svg, avatar1.svg ... avatar5.svg

To embed your own Google Form, the contact.html already includes your form embedded.
To add photos, replace avatar SVG files with JPG/PNG files and update src paths in team.html.
